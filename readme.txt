== Description ==

このプラグインを使うと多言語フォームの作成と申込データの管理ができます。申込データは株式会社パイプドビッツが提供する SPIRAL(R)version2 上で安全に管理されます。

== Installation ==

SPIRAL Form は WordPress ダッシュボード左サイドバーの「プラグイン」メニューにある「新規追加」をクリックし、「SPIRAL Form」を検索してインストールボタンをクリックすることで自動的にインストールできます。 
または以下の手順でもインストールできます。
1. "spiral-form.zip"を解凍してください。
2. "spiral-form"フォルダをFTPソフト等で "/wp-content/plugins/"ディレクトリにアップロードしてください。
3. Wordpressのプラグインページから有効化。

== What is SPIRAL(R)? ==

SPIRAL(R) は株式会社パイプドビッツが提供する Web アプリケーション構築のプラットフォームです。顧客情報や従業員情報といった情報資産を管理、運用するために多彩な機能を提供しています。開発者は DB、メール、Web 等の機能を組み合わせて様々なアプリケーションを構築できます。
